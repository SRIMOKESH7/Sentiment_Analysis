# Sentiment Analysis

A machine learning web application that classifies text into positive, negative, 
and neutral sentiment using NLP preprocessing techniques.

## Features
- Classifies text sentiment (positive/negative/neutral)
- Processed and trained on 1,000+ user reviews
- Simple web interface for real-time predictions

## Tech Stack
- Python
- Scikit-learn
- HTML, CSS

## How to Run
1. Clone this repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python app.py`
4. Open in browser at `http://localhost:5000`
